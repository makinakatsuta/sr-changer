# スクリーンリーダー切り替えツール (sr-changer) v1.3.1

NVDA・ナレーター・PC-Talker Neo の3つのスクリーンリーダーを切り替えるWindows向けツールです。  
**基本的に管理者権限不要（一般ユーザー権限）で動作し、切り替えに失敗した場合のみUAC（管理者権限の確認）が表示されます。**

> [!IMPORTANT]
> **初めてお使いになる方へ（無音時間について）：**
> スクリーンリーダーの切り替え中、現在のスクリーンリーダーが停止してから次のスクリーンリーダーが起動するまで、**数秒から最大10秒程度の無音時間が発生します。**
> パソコンがフリーズしたわけではありませんので、そのままお待ちください。

## 動作フロー

```
[ツール起動] （設定ファイルがなくても、安全にデフォルト設定でそのまま起動します）
    ↓
[選択ダイアログ] ← ここでスクリーンリーダーを選ぶ
    ↓
[切り替え処理] （失敗時のみ、自動でUACプロンプトが表示され管理者権限で再実行します）
    ↓
[現在のSR停止] ← 一時的に音声が出なくなります（最大10秒の無音時間）
    ↓
[選択したSR起動]
```

## 動作要件

- Windows 11
- 一般ユーザー権限（プロセスの停止・起動に失敗した場合のみ、自動的にUACプロンプトが表示されます）
- 以下のスクリーンリーダーのうち、使用するものがインストール済みであること
  - [NVDA](https://www.nvaccess.org/)
  - Windowsナレーター（OS標準搭載）
  - [PC-Talker Neo](https://www.aok-net.co.jp/)（高知システム開発）

## インストール

事前のインストール作業は不要です。`sr-changer.exe` をそのまま実行できます。

## 選択ダイアログ

ツールを起動すると、まず選択ダイアログが表示されます。本ツールには、キーボードで素早く決定できるアクセシブルな設計に加え、マウス操作を行うロービジョンユーザー向けにフォーカス可能で視認性の高い「起動」ボタンと「キャンセル」ボタンが左右に並んで配置されています。

**【ロービジョン（弱視）に配慮したモダンな高コントラストデザイン】**
- **眩しさを抑えるダークモード:** 背景をダークグレー、文字を明るい白に設定し、目の負担や眩しさを軽減しつつ視認性を最大化しています。
- **大きくて太いフォント:** 全要素に大きめの太字を採用し、小さい文字が判別しにくい方でも読みやすくしています。
- **フォーカス時のダイナミック変化:** キーボードの上下キーで移動した際、現在選択されている項目の**背景が浮き上がり、文字色がゴールド（金色）に変化**します。「起動」および「キャンセル」ボタンもフォーカス時に境界線がゴールドになります。これにより、操作位置が視覚的に一目でわかります。

- 起動直後にラジオボタンに自動でフォーカスが当たるため、**キーボードの上下（↑/↓）キーでスクリーンリーダーをすぐに選択可能**です。
- また、表示される選択肢に対応した**数字キー（1〜9キー、およびテンキー）**を押すだけで、対象のスクリーンリーダーを直接選択・フォーカスすることもできます。
- 選択肢（ラジオボタン）にフォーカスした状態で **Enter（Return）キーを押すか、「起動」ボタンを押すことで、そのスクリーンリーダーの起動が決定**されます。
- 切り替えの開始時、現在のスクリーンリーダーが停止する前に、Windows標準の音声合成 (SAPI) により**「[スクリーンリーダー名]を起動します」という音声案内が流れます**。これにより、切り替え中の無音時間（最大10秒）でもフリーズと誤解せず状況を把握できます。
- **現在起動中のスクリーンリーダー**には「（現在起動中）」と表示されます。
- **現在起動中のスクリーンリーダー**がデフォルトで選択・フォーカスされます（起動していない場合はリストの先頭が選択されます）。
- ツールを終了したい場合は、「キャンセル」ボタンを選択して Enter を押すか、**Esc キー**を押すことで何もせず終了します。
- ダイアログの表示時、多重起動防止機能（ミューテックス）により、すでにダイアログが開いている場合は新しく開かずに終了します。

設定ファイルが存在しなくても、デフォルト設定で問題なく動作します。初回起動時に自動生成されることもないため、管理者権限が必要なフォルダ（Program Files など）に配置しても、起動時のパーミッションエラーや意図しないUACプロンプトの発生を防ぐことができます。


```json
{
  "_comment": "起動するスクリーンリーダーを選択します。表示順を変えたい場合は readers の並び順を変えてください。",
  "_values": "使用可能な値: \"nvda\", \"narrator\", \"pctalker\"",
  "_timeout": "SRの停止を待つ最大秒数 (デフォルト: 10)",
  "readers": ["nvda", "narrator", "pctalker"],
  "timeout_sec": 10
}
```

### 設定例

**NVDAとナレーターだけを表示したい場合：**
```json
{
  "readers": ["nvda", "narrator"]
}
```

**PC-TalkerとNVDAだけを表示したい場合：**
```json
{
  "readers": ["pctalker", "nvda"]
}
```

## ホットキーへの登録（推奨）

AutoHotkey などを利用してショートカットキーに割り当てると便利です。

```ahk
; 例: Ctrl+Alt+S で切り替えダイアログを起動
^!s::Run, "C:\path\to\sr-changer.exe"
```

## コマンドライン引数

管理者権限で直接スクリーンリーダーを指定して起動することもできます（タスクスケジューラ等での利用を想定）。

```
sr-changer.exe -reader nvda
sr-changer.exe -reader narrator
sr-changer.exe -reader pctalker
```

## ビルド方法

[Go](https://go.dev/) がインストールされている環境で以下を実行してください。

1. **一括ビルド（推奨）**:
   `build.bat` をダブルクリックするだけで、サイズが最適化された `sr-changer.exe` のビルドが自動で行われます。

2. **手動ビルド**:
   サイズを最適化（デバッグ情報の削除）し、GUIモードでコンソールを表示させずにビルドするには以下を実行します：
   ```powershell
   go build -ldflags="-s -w -H windowsgui" -o sr-changer.exe main.go
   ```

## 注意事項

- 実行には**一般ユーザー権限**で動作可能です。何らかの理由でプロセスの停止や起動が行えなかった場合のみ、自動的にUACプロンプトが表示されます。
- スクリーンリーダーの終了後、完全に停止するまで最大10秒間待機してから次のスクリーンリーダーを起動します。
- `taskkill /F` による強制終了を使用するため、スクリーンリーダー側の保存確認ダイアログ等は無視されます。
