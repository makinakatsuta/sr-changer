スクリーンリーダー切り替えツール (sr-changer) リリースノート

=========================================
バージョン 1.2 (2026年6月21日リリース)
-----------------------------------------
【変更内容】
・スクリーンリーダーの切り替え時間（無音時間）を短縮するため、プロセスの検出および強制終了処理を従来の外部コマンド（tasklist/taskkill）呼び出しから、Windows API を呼び出す純粋な Go ネイティブ実装に変更しました。
・プロセスの完全終了待機時のチェック周期を 500ms から 100ms へ短縮し、切り替えから再起動までのラグを大幅に低減しました。
・選択ダイアログに、マウスを使用する弱視（ロービジョン）ユーザーへの配慮として、フォーカス可能で視認性の高い「起動」ボタンをビジュアル表示するようにしました。デザイン（ダークモード、フォーカス時のゴールド枠など）は既存の「キャンセル」ボタンと統合されています。
・一部の環境で PowerShell の実行制限ポリシーに抵触する問題を回避するため、PowerShell の実行時に `-ExecutionPolicy Bypass` を追加し、UAC昇格やダイアログの安定起動を保証するようにしました。
・NVDA の停止処理について、非管理者権限と管理者権限の両プロセスで統一した停止シークエンス（クリーン終了 `-q` の試行後、失敗時は強制終了）を実行するように改善しました。

バージョン 1.1 (2026年6月20日リリース)
-----------------------------------------
【変更内容】
・リリース用ワークフローの簡素化を図り、配布形式からZIPアーカイブの作成および参照を廃止しました。
・READMEやビルドスクリプト（build.bat）内のZIP作成に関する言及を削除し、sr-changer.exe単体での配布・実行を基本とする形に整理しました。

=========================================
バージョン 1.0 (2026年6月19日リリース) - 初期リリース
-----------------------------------------
【変更内容】
・スクリーンリーダー切り替えツール「sr-changer」の初期バージョンを公開しました。
・NVDA、Windowsナレーター、PC-Talker Neo の3つのスクリーンリーダーをスムーズに切り替えられる機能を実装。
・通常は一般ユーザー権限（管理者権限不要）で動作し、プロセスの切り替えに失敗した場合のみ自動でUAC（管理者権限）を要求して再実行する仕組みを導入しました。
・ロービジョン（弱視）のユーザーに配慮し、ダークモードの配色、拡大表示、太字フォント、キーフォーカス時の色変化（ゴールド）など視認性を最大化したモダンなUIを実装しました。
・開発者向けのサイズ最適化オプション付きの一括ビルドスクリプト（build.bat）を同梱しました。
